Cadastrar pets:
  • Para cadastrar um novo pet siga o passo de buscar um cliente.
  • Clique em detalhes para acessar a página com todos dados do cliente.
  • Clique em novo pet para ser direcionado a página de cadastro do pet.
  • Clique em salvar para registar um novo pet vinculado a este cliente.
  • Clique em cancelar para voltar a página anterior.
Editar pets:
  • Para editar pet siga o passo de buscar um cliente.
  • Clique em detalhes para acessar a página com todos dados do cliente.
  • Selecione o pet a ser atualizado e clique em editar.
  • Clique em atualizar para salvar.
  • Clique em cancelar para voltar a página anterior.
Deletar pets:
  • Para deletar um pet siga o passo de buscar um cliente.
  • Clique em detalhes para acessar a página com todos dados do cliente.
  • Selecione o pet a ser excluído e clique em deletar.
  • Mesmo sem pets, o cadastro do cliente continua em sistema.
  • Siga o passo para deletar cliente, se o cliente deseja apagar todos os dados.
Buscar pets:
  • Para buscar um pet, clique em buscar(azul claro) no menu esquerdo para acessar a página de buscas.
  • Selecione ao menos um campo para busca.
  • É possível atualizar e deletar pets nessa página.
QR code:
  • Para gerar um qr code clique em qrcode(preto) no menu esquerdo.
  • Selecione ao menos um campo para busca.
  • Caso não tenha o código do pet, este pode ser obtido nos detalhes do cliente/pet.
  • É possivel salvar uma imagem do qr code no botão download.
