Clientes

Cadastrar clientes:
  • Para cadastrar um cliente, clique em cadastrar para ser redirecionado a página de cadastro.
  • Os dados informados devem ser do cliente dono do pet.
  • Preencha todos os dados obrigatórios.
  • Um novo cliente pode ser cadastrado mesmo sem ter um pet.
Editar clientes:
  • Para editar um cliente, clique em buscar para ser redirecionado a página de buscas.
  • Faça uma busca do cliente a ser editado, preencha ao menos um campo.
  • Clique em editar para ser redirecionado a página de edição.
  • Ao enviar os dados para edição, certifique-se que não há campos importantes vazios.
Deletar clientes:
  • Para deletar um cliente, clique em buscar para ser redirecionado a página de buscas.
  • Faça uma busca do cliente a ser deletado, preencha ao menos um campo.
  • Clique em deletar para apagar os dados do cliente.
Buscar clientes:
  • Para buscar um cliente, clique em buscar para ser redirecionado a página de buscas.
  • Telefone aceita apenas números.
  • O campo de nome, busca por primeiro nome e segundo nome.
  • Não é necessário colocar @ no campo instagram.
